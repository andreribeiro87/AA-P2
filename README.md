# AA Projeto 2 - Maximum Weight Clique

## Implementação de algoritmos randomized e outros para o problema MWC

### Algoritmos implementados

- Random Construction
- Random Greedy Hybrid
- Iterative Random Search
- Monte Carlo
- Las Vegas
- MWC Peel
- MWC Redu
- Max Clique Weight
- Max Clique Dyn Weight
- WLMC
- TSM MWC
- Fast WClq
- SCC Walk
- Mwc Peel

### Estrutura do projeto

- `src/algorithms.py`: Implementação dos algoritmos
- `src/benchmark.py`: Implementação do benchmark
- `src/graph.py`: Implementação do grafo
- `src/utils.py`: Implementação de utilidades
- `src/visualization.py`: Implementação de visualizações

### Como executar o projeto

Segue a mesma ideia que o projeto passado.
